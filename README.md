# OneCal(カレンダー統合管理アプリ )

## 概要
「OneCal」は、日々の予定管理と家計管理を
カレンダーを軸に統合 したWebアプリです。
情報の分散を防ぎ、日付単位での振り返りを可能にしました。

## 主な機能
- ユーザー登録 / ログイン
- ユーザー編集
- 予定の登録/編集/削除
- 収支の登録/編集/削除
- カレンダー
- 当日詳細画面

## 開発環境
- 言語：PHP
- フレームワーク：Laravel
- データベース：MySQL
- 開発環境：XAMPP
- 本番環境：Heroku
- バージョン管理：Git / GitHub
